# Шаг 16. Стили в head

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 15](15-css-neskolko-svoystv.md) | [Дальше: Шаг 17](17-css-dva-sposoba.md)

## Цель

Собрать все правила оформления в одном месте - в `<head>`. Тогда одно правило сможет покрасить сразу все одинаковые теги.

## Новый прием

Есть второй способ добавить CSS. В `<head>` ставят тег `<style>` и пишут внутри правила.

Одно правило выглядит так:

```html
h1 {
  color: blue;
}
```

Это значит: "все заголовки `h1` сделать синими". Не один тег, а **сразу все** `h1` на странице!

Так же можно задать правило для всех абзацев `p`.

## Что изменить

1. Убери `style="..."` из тегов. Пусть снова будут простые `<h1>` и `<p>`.
2. В `<head>` добавь тег `<style>` с правилами:

```html
<head>
  <meta charset="UTF-8">
  <title>Ничего себе!</title>
  <style>
    h1 {
      color: blue;
      text-align: center;
    }
    p {
      font-size: 20px;
    }
  </style>
</head>
```

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

## Проверка

Открой страницу.

Все заголовки `h1` стали синими и встали посередине. Все абзацы `p` стали крупнее.

Получился тот же результат, что и раньше, но правил в тегах больше нет - они все собраны в `<head>`.

## Маленькое задание

Добавь в `<style>` правило для заголовков раздела `h2`:

```html
h2 {
  color: green;
}
```

Все заголовки `h2` сразу станут зелеными. Видишь, как удобно - одно правило, а покрасились все.
