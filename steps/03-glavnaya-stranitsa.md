# Шаг 3. Главная страница

## Цель

Страница больше не просто сообщает, что сайт в разработке. Теперь это первая версия главной страницы сайта **"Ничего себе!"**.

## Новый прием

Мы можем менять текст внутри уже знакомых тегов.

## Что изменить

Замени текст заголовка:

```html
<h1>Ничего себе!</h1>
```

И оставь два абзаца о сайте.

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Ничего себе!</h1>
    <p>Мир интереснее, чем кажется.</p>
    <p>На этом сайте будут маленькие открытия: про природу, космос, животных, людей и вещи вокруг нас.</p>
  </body>
</html>
```

</details>

## Проверка

Открой страницу в браузере.

Теперь видно, как называется сайт и о чем он будет.

## Маленькое задание

Добавь третий абзац с фразой:

```html
<p>Каждый факт должен вызывать мысль: ничего себе!</p>
```
