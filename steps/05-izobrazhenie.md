# Шаг 5. Изображение

## Цель

Добавить первую картинку на сайт.

## Новый прием

Тег `<img>` показывает изображение.

У него есть важные части:

- `src` - путь к картинке;
- `alt` - текст, который объясняет, что на картинке.

## Подготовка

Создай папку `images`.

Положи в нее картинку и назови файл:

```text
planet.jpg
```

Если картинки пока нет, можно оставить код и добавить файл позже.

## Что добавить

После главного заголовка добавь:

```html
<img src="images/planet.jpg" alt="Яркая планета в космосе">
```

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Ничего себе!</h1>
    <img src="images/planet.jpg" alt="Яркая планета в космосе">
    <p>Мир интереснее, чем кажется.</p>

    <h2>Что здесь будет?</h2>
    <p>На этом сайте будут маленькие открытия: про природу, космос, животных, людей и вещи вокруг нас.</p>

    <h2>Первое открытие</h2>
    <p>Каждый факт должен вызывать мысль: ничего себе!</p>
  </body>
</html>
```

</details>

## Проверка

Открой страницу.

Если картинка лежит в правильной папке, картинка появится под заголовком.

## Маленькое задание

Поменяй текст в `alt`, чтобы он точнее описывал твою картинку.
