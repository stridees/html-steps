# Шаг 9. Список

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 8](08-ssylka-nazad.md) | [Дальше: Шаг 10](10-forma-pole-vvoda.md)

## Цель

Добавить список тем, про которые будет сайт.

## Новый прием

Тег `<ul>` создает список.

Тег `<li>` создает один пункт списка.

## Что добавить

В файле `index.html` после заголовка `Что здесь будет?` добавь список:

```html
<ul>
  <li>Космос</li>
  <li>Природа</li>
  <li>Изобретения</li>
</ul>
```

## Что должно получиться в главной части `index.html`

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<h2>Что здесь будет?</h2>
<ul>
  <li>Космос</li>
  <li>Природа</li>
  <li>Изобретения</li>
</ul>
<p>На этом сайте будут маленькие открытия: про природу, космос, животных, людей и вещи вокруг нас.</p>
```

</details>

## Проверка

На странице должен появиться список с точками.

## Маленькое задание

Добавь четвертый пункт:

```html
<li>Человек</li>
```
