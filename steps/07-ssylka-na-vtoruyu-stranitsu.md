# Шаг 7. Ссылка на вторую страницу

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 6](06-vtoraya-stranitsa.md) | [Дальше: Шаг 8](08-ssylka-nazad.md)

## Цель

С главной страницы можно перейти на страницу с фактами.

## Новый прием

Тег `<a>` создает ссылку.

У ссылки есть `href`. В `href` пишут, куда ведет ссылка.

## Что добавить

В файле `index.html` после первого абзаца добавь:

```html
<p><a href="facts.html">Перейти к удивительным фактам</a></p>
```

## Что должно получиться в `index.html`

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Ничего себе!</h1>
    <img src="images/planet.jpg" alt="Яркая планета в космосе">
    <p>Мир интереснее, чем кажется.</p>
    <p><a href="facts.html">Перейти к удивительным фактам</a></p>

    <h2>Что здесь будет?</h2>
    <p>На этом сайте будут маленькие открытия: про природу, космос, животных, людей и вещи вокруг нас.</p>

    <h2>Первое открытие</h2>
    <p>Каждый факт должен вызывать мысль: ничего себе!</p>
  </body>
</html>
```

</details>

## Проверка

Открой `index.html` и нажми на ссылку.

Браузер должен открыть `facts.html`.

## Маленькое задание

Поменяй текст ссылки на:

```html
Все факты
```
