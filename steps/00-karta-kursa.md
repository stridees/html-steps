# Первый этап: карта курса

Проект: сайт **"Ничего себе!"**

Слоган: **"Мир интереснее, чем кажется"**

На этом этапе мы учимся делать простую HTML-страницу. Каждый шаг маленький: добавляем одну новую вещь и сразу смотрим результат в браузере.

## Что понадобится

- папка проекта;
- файл `index.html`;
- браузер;
- редактор кода.

## Правило микрошагов

На одном шаге мы не стараемся сделать красивый сайт целиком. Мы делаем одно маленькое изменение:

- добавили заголовок;
- проверили;
- добавили абзац;
- проверили;
- добавили картинку;
- проверили.

Так сайт растет спокойно и понятно.

В некоторых шагах есть блок **"Что должно получиться"**. Он спрятан под спойлером. Сначала попробуй сделать шаг сам. Если стало трудно, открой спойлер, подсмотри, закрой его и попробуй повторить по памяти.

## Что получится в конце

У нас будет маленький сайт из двух страниц:

- главная страница `index.html`;
- страница `facts.html`;
- картинка;
- ссылки между страницами;
- список интересных тем;
- простая форма для идеи читателя.
