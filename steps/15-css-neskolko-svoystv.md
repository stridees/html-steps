# Шаг 15. Несколько правил сразу

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 14](14-css-razmer-bukv.md) | [Дальше: Шаг 16](16-css-stili-v-head.md)

## Цель

Дать заголовку сразу несколько правил и поставить его посередине.

## Новый прием

В одном `style` можно написать несколько правил. Их разделяют точкой с запятой `;`.

Новое правило - `text-align`. Значение `center` ставит текст **посередине**.

## Что изменить

Сейчас у заголовка одно правило:

```html
<h1 style="color: blue;">Ничего себе!</h1>
```

Добавь второе правило через точку с запятой:

```html
<h1 style="color: blue; text-align: center;">Ничего себе!</h1>
```

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<h1 style="color: blue; text-align: center;">Ничего себе!</h1>
<img src="images/planet.jpg" alt="Яркая планета в космосе">
<p style="font-size: 20px;">Мир интереснее, чем кажется.</p>
```

</details>

## Проверка

Открой страницу.

Заголовок синий и стоит посередине.

## Маленькое задание

Добавь третье правило - размер заголовка:

```html
<h1 style="color: blue; text-align: center; font-size: 48px;">Ничего себе!</h1>
```

Так в одном теге можно собрать сколько хочешь правил.
