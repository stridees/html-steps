# Шаг 10. Форма: поле ввода

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 9](09-spisok.md) | [Дальше: Шаг 11](11-forma-radioknopki.md)

## Цель

Добавить форму, куда читатель может написать свою идею для сайта.

## Новый прием

Тег `<form>` создает форму.

Тег `<input>` создает поле ввода.

Тег `<label>` подписывает поле, чтобы было понятно, что писать.

## Что добавить

Внизу `index.html`, перед закрывающим тегом `</body>`, добавь:

```html
<h2>Предложи идею</h2>

<form>
  <label for="idea">О чем сделать факт?</label>
  <input id="idea" name="idea" type="text">
</form>
```

## Проверка

На странице появится поле, в которое можно печатать текст.

Форма пока никуда не отправляет данные. Сейчас нам важно научиться показывать поле на странице.

## Маленькое задание

Поменяй подпись поля на:

```html
Какой факт тебя удивил?
```
