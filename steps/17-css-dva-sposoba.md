# Шаг 17. Два способа оформления

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 16](16-css-stili-v-head.md)

## Цель

Понять, чем отличаются два способа добавить CSS и когда какой выбрать. А заодно оформить вторую страницу.

## Новый прием

Мы научились двум способам.

Способ 1 - прямо в теге (`style="..."`):

- красит только **один** тег;
- удобно, когда надо быстро поменять что-то одно.

Способ 2 - в `<head>` (тег `<style>`):

- одно правило красит **сразу все** одинаковые теги на странице;
- удобно, когда хочешь, чтобы все заголовки или все абзацы выглядели одинаково;
- это обычный способ для оформления целой страницы.

Простое правило для выбора:

- много одинаковых тегов - пиши в `<head>`;
- один маленький кусочек - можно прямо в теге.

## Что сделать

Сейчас оформление есть только на главной странице. Вторая страница `facts.html` все еще без стилей.

Скопируй тег `<style>` из `index.html` в `<head>` файла `facts.html`.

## Что должно получиться в `facts.html`

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Факты - Ничего себе!</title>
    <style>
      h1 {
        color: blue;
        text-align: center;
      }
      p {
        font-size: 20px;
      }
    </style>
  </head>
  <body>
    <h1>Удивительные факты</h1>
    <p>Здесь будут факты, после которых хочется сказать: ничего себе!</p>
    <p>Первый факт скоро появится на этой странице.</p>
    <p><a href="index.html">Вернуться на главную</a></p>
  </body>
</html>
```

</details>

## Проверка

Открой `facts.html`.

Теперь обе страницы оформлены одинаково: синий заголовок посередине и крупный текст.

## Маленькое задание

Поменяй цвет в `<style>` только на странице `facts.html`, например на `purple`.

Заметь: страницы стали разными. У каждой страницы свой `<style>`, поэтому каждую можно оформить по-своему.
