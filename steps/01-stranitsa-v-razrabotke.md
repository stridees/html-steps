# Шаг 1. Страница в разработке

[Оглавление](00-karta-kursa.md) | [Дальше: Шаг 2](02-slogan-v-abzatse.md)

## Цель

Создать первую HTML-страницу. Пока сайт еще не готов, поэтому на странице будет только сообщение:

**Сайт находится в разработке**

## Новый прием

HTML-страница похожа на коробку с важными частями:

- `<!doctype html>` говорит браузеру: это современная HTML-страница;
- `<html>` хранит всю страницу;
- `<head>` хранит служебную информацию;
- `<title>` задает название вкладки;
- `<body>` хранит все, что видно на странице;
- `<h1>` делает главный заголовок.

## Что сделать

Создай файл `index.html` и вставь в него код:

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Сайт находится в разработке</h1>
  </body>
</html>
```

## Проверка

Открой `index.html` в браузере.

Ты должен увидеть большой заголовок:

**Сайт находится в разработке**

## Маленькое задание

Поменяй текст в `<title>` на:

```html
<title>Мой первый сайт</title>
```

Посмотри, как изменилось название вкладки в браузере.
