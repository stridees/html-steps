# Шаг 4. Второй заголовок

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 3](03-glavnaya-stranitsa.md) | [Дальше: Шаг 5](05-izobrazhenie.md)

## Цель

Добавить на главную страницу маленький раздел.

## Новый прием

Тег `<h2>` делает заголовок второго уровня. Он меньше, чем `<h1>`, и подходит для раздела страницы.

`<h1>` - главный заголовок всей страницы.

`<h2>` - заголовок части страницы.

## Что добавить

После первого абзаца добавь:

```html
<h2>Что здесь будет?</h2>
```

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Ничего себе!</h1>
    <p>Мир интереснее, чем кажется.</p>

    <h2>Что здесь будет?</h2>
    <p>На этом сайте будут маленькие открытия: про природу, космос, животных, людей и вещи вокруг нас.</p>
    <p>Каждый факт должен вызывать мысль: ничего себе!</p>
  </body>
</html>
```

</details>

## Проверка

На странице теперь два размера заголовков:

- самый главный заголовок сайта;
- заголовок раздела.

## Маленькое задание

Добавь еще один заголовок второго уровня:

```html
<h2>Первое открытие</h2>
```

Поставь его перед самым последним абзацем.
