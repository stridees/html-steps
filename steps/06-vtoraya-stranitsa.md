# Шаг 6. Вторая страница

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 5](05-izobrazhenie.md) | [Дальше: Шаг 7](07-ssylka-na-vtoruyu-stranitsu.md)

## Цель

Создать вторую страницу сайта.

Главная страница называется `index.html`.

Новая страница будет называться `facts.html`.

## Новый прием

Сайт может состоять не из одного файла, а из нескольких HTML-страниц.

## Что сделать

Создай файл `facts.html` рядом с `index.html`.

Вставь в него код:

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Факты - Ничего себе!</title>
  </head>
  <body>
    <h1>Удивительные факты</h1>
    <p>Здесь будут факты, после которых хочется сказать: ничего себе!</p>
  </body>
</html>
```

## Проверка

Открой `facts.html` в браузере.

Ты должен увидеть отдельную страницу с заголовком:

**Удивительные факты**

## Маленькое задание

Добавь второй абзац:

```html
<p>Первый факт скоро появится на этой странице.</p>
```
