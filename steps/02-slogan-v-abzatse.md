# Шаг 2. Слоган в абзаце

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 1](01-stranitsa-v-razrabotke.md) | [Дальше: Шаг 3](03-glavnaya-stranitsa.md)

## Цель

Добавить под заголовок короткую фразу.

## Новый прием

Тег `<p>` создает абзац. Абзац нужен для обычного текста.

## Что изменить

Внутри `<body>` добавь строку с абзацем:

```html
<p>Мир интереснее, чем кажется.</p>
```

## Что должно получиться

Сначала попробуй сделать сам. Открывай этот блок, если стало трудно.

<details>
<summary>Подсмотреть готовый вариант</summary>

```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Ничего себе!</title>
  </head>
  <body>
    <h1>Сайт находится в разработке</h1>
    <p>Мир интереснее, чем кажется.</p>
  </body>
</html>
```

</details>

## Проверка

На странице теперь есть:

- большой заголовок;
- обычная строка текста под ним.

## Маленькое задание

Добавь еще один абзац:

```html
<p>Скоро здесь появятся удивительные факты.</p>
```
