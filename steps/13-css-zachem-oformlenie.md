# Шаг 13. Зачем менять оформление

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 12](12-forma-chekboksy.md) | [Дальше: Шаг 14](14-css-razmer-bukv.md)

## Цель

Сайт уже работает, но выглядит скучно: все буквы черные и одинаковые. Заголовок почти не отличается от обычного текста.

Теперь займемся оформлением. Начнем с заголовка: сделаем его ярким, чтобы он сразу бросался в глаза.

## Новый прием

Раньше мы писали, **что** есть на странице: заголовок, абзац, картинка. Это HTML.

Теперь мы будем говорить, **как это выглядит**: какого цвета буквы, какого размера. Это называется **CSS**.

Проще всего представить так:

- HTML - это слова, картинки и ссылки на странице;
- CSS - это их внешний вид.

Слова могут остаться теми же, а выглядеть страница будет по-разному.

Самый простой способ - добавить прямо в тег слово `style` и написать правило. Первое правило - цвет букв: `color`.

## Что изменить

В файле `index.html` найди главный заголовок:

```html
<h1>Ничего себе!</h1>
```

Поменяй его на:

```html
<h1 style="color: blue;">Ничего себе!</h1>
```

`color: blue;` значит "цвет букв - синий".

## Проверка

Открой `index.html` в браузере.

Главный заголовок должен стать **синим**.

## Маленькое задание

Попробуй другие цвета. Просто меняй слово после `color`:

```html
<h1 style="color: green;">Ничего себе!</h1>
```

Попробуй `green`, `red`, `orange`, `purple`. Выбери цвет, который тебе нравится.
