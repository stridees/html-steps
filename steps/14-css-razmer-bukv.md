# Шаг 14. Размер букв

[Оглавление](00-karta-kursa.md) | [Назад: Шаг 13](13-css-zachem-oformlenie.md) | [Дальше: Шаг 15](15-css-neskolko-svoystv.md)

## Цель

Сделать слоган крупнее, чтобы его было приятнее читать.

## Новый прием

Новое правило - `font-size`. Оно меняет размер букв.

Размер пишут в пикселях: `px`. Пиксель - это крошечная точка на экране. Чем больше число, тем крупнее буквы.

## Что изменить

В файле `index.html` найди абзац со слоганом:

```html
<p>Мир интереснее, чем кажется.</p>
```

Поменяй его на:

```html
<p style="font-size: 20px;">Мир интереснее, чем кажется.</p>
```

## Проверка

Открой страницу.

Слоган стал крупнее обычного текста.

## Маленькое задание

Меняй число и смотри, что получается:

```html
<p style="font-size: 28px;">Мир интереснее, чем кажется.</p>
```

Попробуй `16px`, `20px`, `28px`, `40px`. Найди размер, который тебе нравится.
