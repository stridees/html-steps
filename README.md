# HTML Steps

Курс по HTML, созданный по принципу микрошагов.

Учебный проект постепенно превращается в сайт **"Ничего себе!"** со слоганом:
**"Мир интереснее, чем кажется"**.

Первый шаг курса находится в папке [steps](steps).

## Как проходить

1. Открывайте шаги по порядку.
2. На каждом шаге меняйте только то, что просит файл.
3. После каждого шага открывайте `index.html` в браузере и смотрите, что изменилось.

## Первый уровень

- обязательные теги HTML-страницы;
- заголовки;
- абзацы;
- изображения;
- ссылки;
- списки;
- простые формы: поле ввода, радиокнопки, чекбоксы.
